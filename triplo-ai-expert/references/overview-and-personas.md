# Triplo AI Overview and Personas

## What is Triplo AI?

Triplo AI is a **system-wide AI assistant** that works across any application on your desktop and mobile devices without requiring individual app integrations.

### Key Differentiators

| Feature | What It Means | Why It Matters |
|---------|---------------|----------------|
| **System-wide** | Works in any app or browser | No need for per-app integrations |
| **No Integrations Required** | Reads content directly from your screen | Instant access to context without setup |
| **Multi-platform** | Desktop (Windows/Mac) + Mobile (iOS/Android) | Consistent experience across all devices |
| **Multi-model Support** | Native models + OpenRouter + Bring Your Own Keys (BYOK) | Choose the best model for each task |
| **High Token Allowance** | 2M+ tokens per device per month | Heavy usage without constant refills |
| **Context Awareness** | Reads selected text, windows, clipboard | AI sees exactly what you see |
| **Local Model Support** | Run models locally on your hardware | Privacy, offline capability, cost savings |

### Core Features

- **Real-Time Assistance**: Trigger Triplo from anywhere to get instant AI help
- **SmartPrompts**: Pre-built and custom prompt templates for consistent workflows
- **Custom SmartPrompts**: Create your own reusable prompt templates
- **Training / Minds / Knowledge Bases**: Build trusted knowledge bases from your documents
- **Voice Interaction**: Speak to Triplo and have it speak back
- **Automations**: Send structured JSON to webhooks and APIs
- **Agents (BETA)**: AI agents that can perform multi-step tasks
- **Local LLMs**: Run models locally for privacy and offline use
- **Translations**: Multi-language support
- **Token Allowances**: Generous monthly token limits per device

## Key Value Propositions

### 1. "No Integrations" = Instant Value
Unlike tools that require setting up connections for each app, Triplo works out of the box across your entire system. Select text in any app → Trigger Triplo → Get AI assistance.

### 2. Context Awareness = Better AI
Triplo sees what you see. Whether it's a support ticket in your help desk, code in your IDE, or a research paper in your PDF viewer, Triplo has full context.

### 3. Knowledge Bases = Trustworthy AI
Train Triplo on your documents, policies, and knowledge base. This grounds responses in your actual information, not generic web knowledge.

### 4. Automations = Workflow Power
Send structured JSON from Triplo to any webhook-compatible tool. Create end-to-end workflows: on-screen content → AI processing → multi-app automation.

### 5. Local Models = Privacy + Savings
Run models locally for sensitive work, offline capability, or to save on API costs. Triplo routes to local models seamlessly.

### 6. Multi-model Flexibility
Choose the right model for each task:
- Fast/cheap models for quick tasks
- Powerful models for complex reasoning
- Local models for privacy
- Specialized models for specific domains

## Common Misconceptions

### ❌ "Triplo is just another ChatGPT wrapper"
**Reality**: Triplo is a system-wide assistant with context awareness, automations, knowledge bases, and local model support that goes far beyond simple chat interfaces.

### ❌ "I need to integrate it with every app I use"
**Reality**: Triplo works system-wide without per-app integrations. It reads content directly from your screen.

### ❌ "It's only for desktop"
**Reality**: Triplo works on both desktop (Windows/Mac) and mobile (iOS/Android) with consistent features.

### ❌ "I have to pay extra for each model"
**Reality**: Triplo includes generous token allowances for native models. BYOK lets you use your own OpenAI/OpenRouter keys if needed.

### ❌ "My data is sent to third parties"
**Reality**: You can use local models for complete privacy. Even with cloud models, Triplo has clear data handling policies.

## When Triplo Shines

**Use Triplo when you need to**:
- Process content from multiple apps without copy-pasting
- Build workflows that span multiple applications
- Ensure AI responses are grounded in your own knowledge
- Automate content creation from on-screen sources
- Work offline or with sensitive data (local models)
- Choose different AI models for different tasks
- Have consistent AI assistance across all your devices

**Consider alternatives when you need**:
- Web-only access without desktop installation
- Specific per-app integrations that Triplo doesn't support
- A simple chat interface without system-wide features
