# Getting Started: Installation, Activation, and Setup

## Installation

### Supported Platforms

| Platform | Status | Notes |
|----------|--------|-------|
| **Windows** | ✅ Full Support | Desktop application |
| **macOS** | ✅ Full Support | Desktop application |
| **iOS** | ✅ Full Support | App Store |
| **Android** | ✅ Full Support | Google Play Store |

### Desktop Installation

**Windows**:
1. Download Triplo AI installer from [triplo.ai](https://triplo.ai)
2. Run the installer (.exe file)
3. Follow the installation wizard
4. Launch Triplo AI from Start Menu or desktop shortcut

**macOS**:
1. Download Triplo AI DMG from [triplo.ai](https://triplo.ai)
2. Open the DMG file
3. Drag Triplo AI to Applications folder
4. Launch Triplo AI from Applications

### Mobile Installation

**iOS**:
1. Open App Store
2. Search for "Triplo AI"
3. Download and install
4. Open app and sign in

**Android**:
1. Open Google Play Store
2. Search for "Triplo AI"
3. Download and install
4. Open app and sign in

## Activation

### Account Creation

1. **Sign Up**:
   - Visit [triplo.ai](https://triplo.ai)
   - Click "Get Started" or "Sign Up"
   - Create account with email or social login

2. **Choose Plan**:
   - **Free Tier**: Basic features with limited tokens
   - **Pro Tier**: Full features with generous token allowances
   - **Enterprise**: Custom solutions for teams

### Device Activation

**Desktop**:
1. Open Triplo AI application
2. You'll be prompted to sign in
3. Enter your credentials
4. Device will be automatically associated with your account
5. Your token allowance (e.g., 2M tokens/month) will be available

**Mobile**:
1. Open Triplo AI app
2. Sign in with your account credentials
3. Device will be linked to your account
4. Access your token allowance and settings

### Device Management

**Managing Multiple Devices**:
- Triplo allows multiple devices per account
- Each device has its own token allowance
- View all linked devices in your account settings

**Removing a Device**:
1. Go to account settings on [triplo.ai](https://triplo.ai)
2. Navigate to "Devices" section
3. Select the device to remove
4. Confirm removal

**Device Limits**:
- Free tier: Limited number of devices
- Pro tier: Higher device limit
- Enterprise: Unlimited devices

## Subscription and Billing

### Subscription Plans

| Plan | Monthly Tokens | Devices | Features |
|------|----------------|---------|----------|
| **Free** | Limited | 1-2 | Basic features, limited models |
| **Pro** | 2M+ per device | 5+ | All features, all models, automations |
| **Enterprise** | Custom | Unlimited | Custom solutions, priority support |

### Upgrading Your Plan

1. Log in to your account at [triplo.ai](https://triplo.ai)
2. Go to "Billing" or "Subscription"
3. Select "Upgrade" or "Change Plan"
4. Choose your desired plan
5. Complete payment
6. New features and token allowances activate immediately

### Refunds and Cancellations

**Refund Policy**:
- Check current refund policy on [triplo.ai](https://triplo.ai)
- Refunds typically available within a specified period
- Contact support for refund requests

**Cancelling Subscription**:
1. Go to account settings
2. Navigate to "Billing"
3. Select "Cancel Subscription"
4. Follow cancellation process
5. Access continues until billing period ends

## Initial Setup

### First-Time Configuration

**After Installation and Activation**:

1. **Choose Your Default Model**:
   - Select from available native models
   - Or configure BYOK (Bring Your Own Keys) for OpenAI/OpenRouter
   - See [`models-allowances-and-byok.md`](models-allowances-and-byok.md) for guidance

2. **Set Up Hotkeys** (Desktop):
   - Configure keyboard shortcuts to trigger Triplo
   - See [`ui-triggering-and-hotkeys.md`](ui-triggering-and-hotkeys.md) for details

3. **Explore SmartPrompts**:
   - Browse built-in SmartPrompts library
   - Try a few to understand the workflow
   - See [`smartprompts-and-custom-smartprompts.md`](smartprompts-and-custom-smartprompts.md) for more

4. **Optional: Set Up Training/Knowledge Base**:
   - Upload your documents
   - Create a "Mind" for your knowledge
   - See [`training-and-knowledge-bases.md`](training-and-knowledge-bases.md) for setup

### Customizing Model List Per Device

**Why Customize?**
- Different devices may have different use cases
- Some models work better for specific tasks
- Save costs by using cheaper models where appropriate

**How to Customize**:
1. Open Triplo AI on the device
2. Go to Settings → Models
3. Select which models to enable for this device
4. Set default model for this device
5. Save changes

**Example Configurations**:
- **Work Laptop**: Powerful models for complex tasks
- **Personal Laptop**: Faster, cheaper models for quick tasks
- **Mobile Device**: Optimized models for mobile use

## Troubleshooting Installation

### Common Issues

**Issue: Installer won't run**
- **Solution**: Run as administrator (Windows) or check security settings (macOS)
- **Check**: System requirements and compatibility

**Issue: Can't sign in**
- **Solution**: Verify email/password, check internet connection
- **Check**: Account status (not suspended or banned)

**Issue: Device not activating**
- **Solution**: Ensure you're logged in, check device limit
- **Check**: Subscription status and available slots

**Issue: Token allowance not showing**
- **Solution**: Refresh account, check subscription status
- **Check**: Billing is active and payment processed

**Issue: Multiple devices not allowed**
- **Solution**: Upgrade plan or remove unused devices
- **Check**: Device limit for your current plan

### Getting Help

If you encounter issues during installation or activation:

1. Check the FAQ in [`troubleshooting-and-faq.md`](troubleshooting-and-faq.md)
