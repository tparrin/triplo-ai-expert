# Learning Paths and Resources

## Curated Learning Tracks

This guide provides structured learning paths from beginner to power user, pointing to specific doc sections and recommending relevant content.

## Track 1: Beginner (0-2 weeks)

### Goal: Get comfortable with basic Triplo AI usage

**Week 1: Setup and Basics**

**Day 1-2: Installation and Activation**
- Read: [`getting-started-install-activate.md`](getting-started-install-activate.md)
- Install Triplo on your primary device
- Create account and activate device
- Explore the interface

**Day 3-4: Triggering and Basic Usage**
- Read: [`ui-triggering-and-hotkeys.md`](ui-triggering-and-hotkeys.md)
- Set up hotkeys
- Practice triggering Triplo from different apps
- Try basic prompts

**Day 5-7: Context Awareness**
- Read: [`context-awareness-and-sources.md`](context-awareness-and-sources.md)
- Try Context mode (full window)
- Try Selection mode (highlighted text)
- Try Clipboard mode
- Upload a file and process it

**Week 2: SmartPrompts and Basic Workflows**

**Day 8-10: SmartPrompts**
- Read: [`smartprompts-and-custom-smartprompts.md`](smartprompts-and-custom-smartprompts.md)
- Try built-in SmartPrompts
- Create your first custom SmartPrompt
- Test with different inputs

**Day 11-14: Practice Workflows**
- Practice selecting text and processing
- Use SmartPrompts for daily tasks
- Experiment with different models
- Explore voice interaction

**Beginner Milestones**:
- [ ] Installed and activated Triplo
- [ ] Can trigger Triplo from any app
- [ ] Used all awareness modes
- [ ] Created a custom SmartPrompt
- [ ] Processed files and images

## Track 2: Intermediate (2-4 weeks)

### Goal: Master core features and build efficient workflows

**Week 3-4: Advanced SmartPrompts**

**Day 15-18: Custom SmartPrompt Design**
- Read: [`smartprompts-and-custom-smartprompts.md`](smartprompts-and-custom-smartprompts.md) (Advanced section)
- Design SmartPrompts for your workflows
- Create structured output SmartPrompts
- Test and iterate

**Day 19-21: SmartPrompt Organization**
- Organize SmartPrompts by category
- Create naming conventions
- Set up instant hotkeys
- Document your SmartPrompts

**Week 5-6: Training and Knowledge Bases**

**Day 22-25: Training Basics**
- Read: [`training-and-knowledge-bases.md`](training-and-knowledge-bases.md)
- Gather documents for training
- Create your first Mind
- Train and test

**Day 26-28: Advanced Training**
- Create multiple Minds
- Test with different queries
- Refine based on results
- Combine Minds with SmartPrompts

**Week 7-8: Automations**

**Day 29-32: Automation Setup**
- Read: [`automations-webhooks-api-magnets.md`](automations-webhooks-api-magnets.md)
- Choose an automation tool (Activepieces, Zapier, etc.)
- Create your first automation
- Test end-to-end

**Day 33-35: Advanced Automations**
- Create JSON-structured SmartPrompts
- Build multi-step workflows
- Use Magnets for reusable components
- Document your automations

**Intermediate Milestones**:
- [ ] Created 5+ custom SmartPrompts
- [ ] Built a knowledge base with multiple Minds
- [ ] Set up at least one automation
- [ ] Integrated with external tools
- [ ] Organized workflows efficiently

## Track 3: Advanced (4-8 weeks)

### Goal: Become a Triplo AI power user

**Week 9-10: Local Models**

**Day 36-40: Local Model Setup**
- Read: [`local-llms-setup-desktop-mobile.md`](local-llms-setup-desktop-mobile.md)
- Install inference engine (Ollama, etc.)
- Download local models
- Configure Triplo to use local models

**Day 41-42: Local Model Optimization**
- Test different local models
- Optimize for your hardware
- Compare local vs. cloud performance
- Set up mobile access to local models

**Week 11-12: Model Mastery**

**Day 43-46: Model Selection**
- Read: [`models-allowances-and-byok.md`](models-allowances-and-byok.md)
- Read: [`openrouter-models-strengths.md`](openrouter-models-strengths.md)
- Read: [`local-models-strengths.md`](local-models-strengths.md)
- Create model selection guide for your use cases

**Day 47-49: BYOK and Advanced Configuration**
- Set up BYOK with your own keys
- Configure per-device model settings
- Optimize token usage
- Monitor and manage costs

**Week 13-16: Advanced Workflows**

**Day 50-55: Complex Automations**
- Build multi-tool automations
- Create agent-like workflows
- Integrate with multiple services
- Implement error handling

**Day 56-60: Agents and Connections**
- Read: [`agents-and-connections.md`](agents-and-connections.md)
- Set up external connections
- Create and configure Agents
- Test autonomous workflows

**Day 61-63: Advanced Features**
- Explore voice and conversation modes
- Read: [`chat-mode-voice-and-conversation.md`](chat-mode-voice-and-conversation.md)
- Implement advanced context management
- Create complex prompt chains

**Advanced Milestones**:
- [ ] Set up and optimized local models
- [ ] Mastered model selection for different tasks
- [ ] Built complex multi-step automations
- [ ] Configured Agents and Connections
- [ ] Optimized token usage and costs
