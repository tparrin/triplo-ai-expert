# Checklists

This collection provides practical checklists for common Triplo AI tasks and workflows. Use these to ensure you don't miss important steps.

## Getting Started Checklist

### New User Setup

- [ ] Download Triplo AI for your platform
- [ ] Install Triplo AI
- [ ] Create Triplo account
- [ ] Activate your device
- [ ] Choose subscription plan (if applicable)
- [ ] Configure default model
- [ ] Set up hotkeys
- [ ] Try a built-in SmartPrompt
- [ ] Test context awareness modes
- [ ] Explore the interface

### First Week Goals

- [ ] Trigger Triplo from 3+ different apps
- [ ] Use all awareness modes (Context, Selection, Clipboard)
- [ ] Create your first custom SmartPrompt

- [ ] Try voice interaction
- [ ] Read 3 reference documents
- [ ] Complete a beginner learning path

## SmartPrompt Creation Checklist

### Before Creating

- [ ] Define the purpose clearly
- [ ] Identify target audience
- [ ] Determine input format
- [ ] Specify output format
- [ ] List constraints
- [ ] Gather examples

### Creating the SmartPrompt

- [ ] Write clear role definition
- [ ] Provide specific instructions
- [ ] Include examples (few-shot)
- [ ] Define output structure
- [ ] Add constraints
- [ ] Test with sample input
- [ ] Refine based on results
- [ ] Document the SmartPrompt

### After Creating

- [ ] Test with various inputs
- [ ] Verify output quality
- [ ] Check for edge cases
- [ ] Save with descriptive name
- [ ] Add tags for organization
- [ ] Set up hotkey (if needed)
- [ ] Share with team (if applicable)
